Data Contribution

This KIRI HackJak app is made using government's open data. Altough with many datasets (550 tracks+), but the data only as detail as road names. For more advanced usage, a more precise data is needed, which is the latitude/longitude list which can be only inputted from map. With a more precise data, route searching becomes more accurate, as in Smart Public Transport app for Bandung.

How can I help?

Help us migrate each UKP4's trayek data to a non-profit site Peta Angkutan Umum, using the guidelines provided in http://hackjak.kiri.travel. The data you input will help, belongs to public, and can be used for a better app!

Thank you.
